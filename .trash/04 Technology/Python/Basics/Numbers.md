# Operations :

```
10 + 3 # Addition
10 - 3 # Substraction
10 * 3 # Multiplication
10 / 3 # Division, with decimal value -> 3,3333
10 // 3 # Division with integer -> 3
10 % 3 #  Mudulus or reminder of a division -> 1 = 10 - (3*3)
10 ** 3 # Exponent 10^3

```

## Advanced operations

- Operate over same variable, omit variable and add sign.
```
x = 10
x = x + 3
x +=3
```

- Round
```
round(2.9) --> 3
```

- Absolute value
```
abs(-2.9) --> 2.9
```

## Comparison:

```
10 > 3  # Greater than
10 >= 3 # greater or equal
10 < 20 # smaller than
10 <= 20 # Smaller or equal
10 == 10 # Equal than 
10 != 10 # NOT equal
```
